<h1>Distros Linux</h1>

Site informativo desenvolvido em HTML e CSS com o objetivo de apresentar diversas distribuições Linux, seus requisitos, características, ambientes gráficos e informações sobre jogos no Linux utilizando Proton.

https://nicolasfmp.github.io/Tutorial-Linux/

<h2>Sobre o Projeto</h2>

O Distros Linux é um portal estático criado para auxiliar usuários interessados em conhecer o ecossistema Linux. O site reúne informações sobre distribuições populares organizadas por categorias, ambientes de desktop e ferramentas utilizadas para jogos.
